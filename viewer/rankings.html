USAU Rankings Algorithm v2.0

Rating Differential

Each game earns a rating based on the opponent's strength ± a score differential x.

r = L / (W − 1)
x = 125 + 475 × sin(min(1,(1−r)/0.5) × 0.4π)
                  / sin(0.4π)

Score Weight

Low-scoring games count less. Full weight (1.0) when W ≥ 13 or W+L ≥ 19.

score_wt = min(1, √((W + max(L, ⌊(W−1)/2⌋)) / 19))

Date Weight

Recent games count more. Exponential interpolation from 0.5 (first week) to 1.0 (last week).

t = days_elapsed / season_days
date_wt = 0.5 × 2t

Iteration

All teams start at 1000. Game ratings are recomputed using each iteration's ratings until convergence.

rating = Σ(game_rating × weight) / Σ(weight)
game_rating = opp_rating ± x

Blowout Ignore Rule

A game is excluded if all three conditions hold, preventing rating inflation from weak opponents.

1. winner_rating − loser_rating > 600
2. W > 2 × L
3. winner has ≥ 5 other valid results
